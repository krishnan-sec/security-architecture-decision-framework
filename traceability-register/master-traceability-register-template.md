## Field Definitions

- **Risk ID** - Reference to enterprise risk register  
- **Decision ID** - Security Architecture Decision (SAD-###)  
- **Control Name** - Implemented technical or administrative control  
- **Domain** - Identity / Network / Cloud / Application / Data / etc.  
- **Control Owner** - Accountable individual or team  
- **Validation Method** - How control effectiveness is verified  
- **Last Review Date** - Most recent validation or decision review  
- **Status** - Active / Exception / Retired  
- **Exception ID** - Linked exception record (if applicable)
